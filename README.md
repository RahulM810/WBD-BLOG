# WBD-BLOG — Cyber Security Awareness Blog

A simple, beginner-friendly website built to educate users on everyday cyber security topics like phishing, ransomware, and password safety.

## About

This is a beginner web development project focused on practicing core HTML/CSS/JS skills while building something useful — a blog-style site that raises awareness about common cyber threats and safe online practices.

## Pages

| Page | Description |
|---|---|
| `index.html` | Homepage / landing page |
| `login.html` | Login page |
| `mission.html` | About the project's mission |
| `team.html` | Team / contributors page |
| `tools.html` | Useful security tools and resources |
| `quiz.html` | Interactive quiz to test cyber security knowledge |
| `password-guidelines.html` | Best practices for creating strong passwords |
| `phishing-alert.html` | Information on identifying and avoiding phishing attacks |
| `zero-day-alert.html` | Awareness content on zero-day vulnerabilities |

## Tech Stack

- HTML
- CSS
- JavaScript (`server.js`)

## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/RahulM810/WBD-BLOG.git
   cd WBD-BLOG
   ```

2. Open `index.html` directly in your browser, **or** serve it locally:
   ```bash
   node server.js
   ```
   > Update this step if `server.js` expects a specific port or additional setup.
